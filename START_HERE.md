## 💡 Dicas Importantes

### 💚 Faça Isso

- ✅ Teste em mobile (80% dos acessos)
- ✅ Otimize imagens (< 500KB cada)
- ✅ Teste todos os links do WhatsApp
- ✅ Faça backup antes de grandes mudanças
- ✅ Use Git para controle de versão

### 🚫 Evite Isso

- ❌ Subir sem testar o WhatsApp
- ❌ Usar imagens muito grandes
- ❌ Esquecer de alterar o número padrão
- ❌ Fazer muitas mudanças de uma vez
- ❌ Pular o teste em mobile
